# CalendarAppBackend
 Backend para la aplicación de calendario.
 Realizado en NodeJS y utilizando MongoDB.

# Funcionalidad
* Permite iniciar y cerrar sesión.
* Crear usuarios.
* CRUD de los eventos del calendario.
